Line Follower Robot / Grid Maze Solver Robot

This project was developed in the context of the course of Embedded Computing Architectures. The goal was to develop a robot that was capable of following a line and solving a grid maze.

Smart Christmas Tree Lights

This project was inspired by a video by the youtuber Stand-up Maths. The goal was to develop a system that would allow to control each led based on its coordinates on the christmas tree. And create effects with the lights that otherwise would not be possible with normal lights.

Visit Project Github Page →

Internet of Lockers

The Internet of Lockers project was developed as part of the Integrated Project for the Degree in Computer Engineering and Computing at the Faculty of Engineering of the University of Porto. The Internet of Lockers aims to facilitate the collection and return of FEUP equipment, allowing users of the reservation system to carry out these actions outside the operating hours of the equipment rental service, using lockers at any time they wish.
